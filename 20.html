Скриншот через bash(fish)

Для того чтобы быстро сделать скриншот сайта не покидая консоль. Надо либо через команду

firefox --screenshot www.vstebunov.me

либо через утилиту cutycurl она есть в репозитории Ubunut откуда её можно установить.

Простой запрос на скриншот сайта выглядит вот так:

cutycapt --max-wait=1200 --url=www.vstebunov.me --out=test.png
